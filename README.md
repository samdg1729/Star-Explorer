# Nearby Stars Explorer (Web App)

A simple Flask-based web application to explore nearby stars within 100 light years.

## Features
- View stars sorted by distance
- Filter stars with potential habitable planets
- Simple web interface

## Technologies Used
- Python
- Flask
- SQLite

## Live Demo

Access the web app here:
https://star-explorer.onrender.com

## How to Run Locally

1. Install dependencies:
   pip install flask

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
